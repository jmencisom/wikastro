# WIKASTRO
Automatic Creation of Wiki Content for Astronomical Objects

## Getting Started

### Prerequisites

* Python 2.7
* pip
* virtualenv (optional)

### Installing

1. Optional: Create a local virtualenv 
```
$ virtualenv wikastro-ve
```
  1.1. Go to the new directory
```
$ cd wikastro-ve
```
  1.2. Load the directory
```
$ source bin/activate
```
2. Clone the repository
```
$ git clone https://github.com/jmencisom/wikastro.git
```
3. Go to directory
```
$ cd wikastro
```
4. Install requirements
```
$ pip install -r requirements.txt
```
